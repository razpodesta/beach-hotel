# Manifiesto: Red de Alianzas y Partner Hub
**Versión:** 1.0 (B2B Growth Engine)
**Estatus:** OBLIGATORIO
**Workspace:** `@metashark/partner-network`

## 1. Misión
Empoderar a las agencias de viajes y aliados comerciales mediante herramientas de personalización de marca blanca (White-Label) y seguimiento de referidos.

## 2. Identidad del Partner (B2B Profile)
- Registro de Logo, Datos de Contacto, Código IATA y redes sociales.
- Generación automática de rumbos de referido: `hotel.com/[lang]/p/[partner_slug]`.

## 3. Flyer Synth (Visual Engine)
- **Integración API:** Conexión con Canvas API o motor interno Sharp.
- **Funcionalidad:** Superposición dinámica del logo del partner sobre los activos de lujo del hotel.
- **Salida:** Generación de PDFs de alta resolución y JPEGs optimizados para redes sociales.

## 4. Reporting de Conversión
- Dashboard privado para cada aliado con métricas de: Clics en link, Reservas generadas y Comisiones acumuladas.
